# 🧬 Evolució Absurda - Projecte Entorns

Aquest és un micro-videojoc desenvolupat per a l'assignatura d'**Entorns de Desenvolupament (2025-26)**. El projecte utilitza la metodologia de **Vibe Coding** per crear una experiència evolutiva dins de Roblox Studio.

## 🎮 El Joc
En aquest simulador d'evolució, el jugador comença com una simple **pedra** i ha d'acumular energia per evolucionar cap a formes de vida més complexes.

### Fases d'Evolució:
1.  **Pedra**: La forma inicial. Recollida d'energia bàsica.
2.  **Planta**: Evolució als 50 punts. Multiplicador x2.
3.  **Animal**: Evolució als 150 punts. Multiplicador x5.
4.  **Humà**: Forma final als 300 punts. Multiplicador x10.

## 🛠️ Tecnologies Utilitzades
*   **Motor**: Roblox Studio
*   **Llenguatge**: Luau (Lua)
*   **Metodologia**: Vibe Coding amb IA
